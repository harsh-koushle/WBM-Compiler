Documentation

Variables & Types

Clarity is a statically typed language, so you must declare the type of a variable before assigning a value. Core types include string, int, and bool.

string message = "Hello from Clarity!";
int version = 1;
bool is_active = true;
print(message);

Conditionals

Control the flow of your program using standard if/else statements. These structures execute different blocks of code based on whether a given condition evaluates to true or false.

int score = 85;
if (score > 50) {
    print("You passed!");
} else {
    print("Please try again.");
}

Loops

Repeatedly execute blocks of code using while and for loops. The for loop is perfect for iterating a known number of times.

print("Counting from 5 to 7:");
for (int j = 5; j < 8; j = j + 1) {
    print(j);
}

Arrays

Arrays are ordered lists that hold values of the same type. You can access and modify elements using zero-based indexing.

int[] numbers = {10, 20, 30, 40};
print(numbers[2]); // Prints 30

numbers[2] = 300;
print(numbers[2]); // Prints 300
